description: >-
  lstr is a fast, minimalist directory tree viewer written in Rust that
  modernizes the classic tree command. It features both a standard command-line
  output and a fully interactive, keyboard-driven TUI mode that supports Nerd
  Font icons, advanced sorting, and real-time Git status integration. Built for
  seamless shell workflows, it natively respects your .gitignore rules, inherits
  your existing system color configurations, and can be easily scripted to
  function as a visual directory switcher.
