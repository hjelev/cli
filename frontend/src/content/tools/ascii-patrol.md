description: >-
  Ascii Patrol is an open-source, terminal-based ASCII game written primarily in C++ that brings the classic retro arcade experience of "Moon Patrol" directly to the command line. The project translates the nostalgic aesthetic of vintage side-scrolling games into a highly interactive text user interface, complete with smooth vehicular combat animations and integrated audio. It offers extensive cross-platform compatibility, allowing it to be compiled natively for Linux, Windows, macOS, DOS, or executed in modern web browsers via Emscripten.
