description: >-
  psmux is a native Windows terminal multiplexer that provides a tmux-like
  experience by leveraging the Windows ConPTY API for seamless session
  management.  

  It enables you to split, rearrange, and persist terminal panes within
  PowerShell, cmd.exe, or Windows Terminal without requiring WSL or external
  compatibility layers.  

  Designed specifically for Windows, it bridges the gap for power users who want
  robust window management and session persistence in their native environment.
