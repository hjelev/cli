description: >-
  vifm is a curses-based, Vim-like file manager that provides a modal,
  keyboard-driven environment for managing your file system.  

  It borrows keybindings, modes, options, registers, and commands from Vim,
  allowing you to navigate and manipulate files without learning an entirely new
  paradigm.  

  Written primarily in C, it adheres to the Unix philosophy and supports
  extensive customization through configuration files like vifmrc.  

  It also features useful ideas inspired by mutt, dual-pane views, and rich
  plugin/preview extensions for images and icons in supported terminals.
