description: >-
  ranger is a free, open-source console file manager with VI keybindings and a
  minimalist curses interface.  

  It uses the Miller columns technique to display the directory hierarchy across
  three columns, showing parent directories, current contents, and file
  previews.  

  It includes rifle, an advanced file launcher that automatically determines
  appropriate applications for different file types.
