  Crush is an intelligent, terminal-based AI coding agent built by the
  Charmbracelet team, designed to integrate seamlessly with your existing
  development workflows, tools, and preferred LLMs.  

  It offers advanced features like multi-model routing with context-preserving
  switching, deep integration with Language Server Protocol (LSP) for code
  awareness, and extensibility via the Model Context Protocol (MCP).  

  With a focus on performance and a "glamorous" TUI, it provides an isolated,
  session-based workspace that works consistently across all major operating
  systems, including Linux, macOS, Windows, and BSD.
