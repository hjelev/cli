short_description: Log file navigator
description: >-
  lnav (The Logfile Navigator) is an advanced, terminal-based log file viewer
  that automatically merges, indexes, and analyzes logs from multiple sources.  

  It simplifies log management by providing features like real-time tailing,
  error highlighting, and structured data extraction, all within a single
  interactive interface.  

  Beyond basic viewing, lnav allows you to perform complex log analysis using
  SQL queries, making it a powerful alternative to standard tools like grep,
  tail, and less.
