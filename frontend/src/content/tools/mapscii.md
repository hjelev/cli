description: >-
  MapSCII is a terminal-based world map renderer that transforms vector tiles
  into interactive Braille and ASCII art.  

  It allows users to pan, zoom, and explore global map data directly in their
  shell, supporting both public vector tile servers and offline local files.  

  You can experience it instantly by connecting via telnet mapscii.me or run it
  locally using Node.js for a smooth, high-performance console experience.
